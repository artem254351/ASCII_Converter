# ASCII Image Converter

Конвертирует изображения в ASCII‑арт прямо в консоли.

## Особенности

* Простой интерфейс командной строки.
* Поддержка разных форматов изображений (JPG, PNG и др.).
* Настройка ширины вывода.
* Обработка пробелов в путях (через кавычки).

## Установка

1. Установите Python (версия 3.8+).
2. Установите зависимости:
```bash
pip install -r requirements.txt
